# Bank Management Application

## Project Overview

The Bank Management Application is a console-based C++ project developed using Object-Oriented Programming (OOP) concepts and File Handling. It allows users to perform basic banking operations such as account creation, deposits, withdrawals, balance inquiries, and account searches.

Customer information is stored permanently in a text file, ensuring data persistence even after the application is closed.

## Features

- Create New Account
- Display All Accounts
- Search Account by Account Number
- Deposit Money
- Withdraw Money
- Check Account Balance
- File Handling for Permanent Storage
- User-Friendly Menu Interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling (fstream)

## Files

- main.cpp
- bank_data.txt (generated automatically)
- README.md
